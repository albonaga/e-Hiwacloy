Key Issues Identified

1

Appointment Rescheduling Confusion

Some respondents had difficulty locating the reschedule and cancel appointment functions.

2

Mobile Responsiveness Issues

Minor layout inconsistencies were observed on smaller mobile devices.

3

Need for Better Instructions

Some respondents required guidance when using the system for the first time.

4

Occasional Slow Loading

A few users experienced delays while accessing specific system features and pages.

Discussion of Findings

Positive Feedback from Users

Easy to Use
Users found the system simple and understandable.
User-Friendly
The interface was comfortable and easy to interact with.
Convenient
Users appreciated faster and easier appointment booking.
Accessible
The system can be easily accessed by users anytime.
Fast Appointment
Booking appointments became quicker and more efficient.
Simple Interface
The layout and design were clean and organized.
Time Saving
Users spent less time processing appointments.
Easy Navigation
Users could easily move through different pages.

Negative Feedback from Users

Limited Features
Users suggested adding more advanced functions.
Notification Delays
Some reminders and alerts appeared late.
Small Text Size
Some users found the text difficult to read.
Confusing Icons
Certain icons were unclear to first-time users.
Slow Loading
Some pages required more loading time.
Mobile Layout Issues
Minor layout problems appeared on smaller screens.
Limited Accessibility
Some features were difficult for certain users to access.
Need Better Instructions
Users requested clearer guides and instructions.
